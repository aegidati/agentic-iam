# App Folder

This repository is documentation-first, but the app folder exists deliberately.

## Why This Folder Exists

Some starter ecosystems expect a stable top-level location for installable or promoted assets. In AGENTIC-IAM, the app folder is reserved for future starter-compatible assets if this foundation later evolves to include installable artifacts that still preserve architecture agnosticism.

## Current State

At this stage, the folder does not contain runtime application code. That is intentional.

## Future Use

If the starter evolves, this folder may host assets such as:

- template fragments intended for controlled adoption
- starter installation helpers that remain stack-neutral
- packaging assets for distribution workflows

Any future additions should preserve the core constraint of this repository: AGENTIC-IAM is a reusable Identity & Access foundation starter, not a runtime stack implementation.
